# AI SQL SaaS

An AI-powered SQL assistant that allows users to query databases using natural language.

## Features
- User authentication (login/register)
- Connect to MySQL / SQLite databases
- Convert natural language → SQL using OpenAI
- Automatic SQL correction
- Data visualization (bar, line, pie charts)

## Tech Stack
- Streamlit
- SQLAlchemy
- OpenAI API
- Plotly
- SQLite / MySQL
